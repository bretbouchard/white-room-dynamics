# Dynamics Presets

## Factory Presets

### Light Compression
- **Purpose**: Gentle dynamic control for mastering
- **Threshold**: -20 dB
- **Ratio**: 2:1
- **Attack**: 20 ms
- **Release**: 200 ms

### Vocal Compression
- **Purpose**: Standard vocal compression
- **Threshold**: -18 dB
- **Ratio**: 3:1
- **Attack**: 10 ms
- **Release**: 150 ms

### Drum Compression
- **Purpose**: Punchy drum compression
- **Threshold**: -15 dB
- **Ratio**: 4:1
- **Attack**: 5 ms
- **Release**: 50 ms

### Peak Limiter
- **Purpose**: Transparent peak limiting
- **Threshold**: -0.1 dB
- **Ratio**: 20:1
- **Attack**: 0.1 ms
- **Release**: 100 ms

### Bass Compression
- **Purpose**: Tight bass guitar compression
- **Threshold**: -12 dB
- **Ratio**: 5:1
- **Attack**: 15 ms
- **Release**: 100 ms

## Creating Custom Presets

Users can create and save custom presets via the host's preset save functionality.

## Preset Formats

- **VST3**: `.vst3preset` files
- **AU**: `.aupreset` files
- **CLAP**: Presets embedded in plugin
- **Standalone**: XML-based preset files
